O mně

Když jsem spustila svůj vlastní e-shop, znala jsem pouze základy HTML a CSS a počítala jsem s tím, že se kódování vyhnu, jelikož jsem využila e-commerce platformu, kde už je vše připravené. Ale zjistila jsem, že ačkoli platforma nabízí solidní řešení, kódování se úplně nevyhnu, už jen kvůli doplnění vizuální identity značky.


Ukázalo se ale, že mě kódování a úpravy webu dost baví, tak jsem se zdokonalila v HTML a CSS a začala se učit programovací jazyk JavaScript, abych mohla pracovat jako front-end vývojářka, a mohla tak sobě i ostatním upravovat weby, aby s nimi mohli co nejlépe oslovit své návštěvníky.
